📋 CHECKLIST DE IMPLEMENTAÇÃO

1. ⚙️ CONFIGURAR FIREBASE
   □ Criar projeto no Firebase Console
   □ Ativar Firestore Database
   □ Copiar credenciais
   □ Colar em src/firebase.config.ts

2. 🔧 EDITAR app.component.ts
   □ Adicionar imports (GameStateService, SaveLoadMenuComponent)
   □ Injetar GameStateService
   □ Adicionar signal showSaveMenu
   □ Adicionar SaveLoadMenuComponent aos imports do @Component
   □ Adicionar métodos saveGame(), loadGame(), autoSaveGame()
   □ Adicionar await this.autoSaveGame() em simulateDivisionRound()

3. 🎨 EDITAR app.component.html
   □ Adicionar botão "Salvar/Carregar" no topo
   □ Adicionar componente <app-save-load-menu>

4. ✅ TESTAR
   □ npm run dev
   □ Clicar em "Salvar/Carregar"
   □ Criar um save
   □ Jogar algumas rodadas
   □ Carregar o save

💡 DICAS

✅ Auto-save salva a cada rodada automaticamente
✅ Saves manuais são úteis antes de decisões importantes
✅ Você pode desativar o auto-save no menu
✅ Os saves ficam salvos na nuvem (Firebase)
✅ Não perca as credenciais do Firebase!
